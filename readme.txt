== Installation ==

> ### First download plugins ZIP file
>
> * For automatic installation:
> * The simplest way to install is to click on 'Plugins' then 'Add' and type 'Photo-Gallery' in the search field
>
> * For manual installation 1:
> * Login to your website and go to the Plugins section of your admin panel
> * Click the Add New button
> * Under Install Plugins, click the Upload link
> * Select the plugin zip file (Photo-Gallery-Responsive.x.x.x.zip) from your computer then click the Install Now button
> * You should see a message stating that the plugin was installed successfully
> * Click the Activate Plugin link
> * For manual installation 2:
> * You should have access to the server where WordPress is installed. If you don't, see your system administrator
> * Copy the plugin zip file up to your server and unzip it somewhere on the file system
> * Copy the "photo-gallery" folder into the /wp-content/plugins directory of your WordPress installation
> * Login to your website and go to the Plugins section of your admin panel
> * Look for photo-gallery plugin and click Activate
>
> ### Or Mac Users
>
> * Go to your Downloads folder and locate the folder with the Juna-IT-gallery-photo plugin
> * Right-click on the folder and select Compress
> * Now you have a newly created .zip file which can be installed as described here.*
> * Click "Install Now" button
> * Click "Activate Plugin" button for activating the gallery plugin

== Frequently Asked Questions ==

### How can I add images to the Gallery?
The images can be added to gallery using its standard File Manager. The uploader can be used both for uploading individual images or multiple image (This should done by uploading a .zip archive file)

What are the best dimensions to set for Images?
We recommend to use images with the width of 1200px to 1600px. In order to speed up the load time of one's site, please use images with size of 500-700 KB or less

Can I add images to my galleries which had already been uploaded to WordPress Media Library?
To enable the feature of adding Media Library images, go to gallery> Options > Global Options and set Import from Media Library option to Yes. After that you will be able to import photos from Media Library.

### How to get the updated version of the Gallery Photo?
Pro users can get updated version of plugin contacting us via the following email address: [junait.com@gmail.com.](mailto:junait.com@gmail.com).

### What is the difference between galleries and albums?
The galleries are designed to contain images, whereas albums contain either galleries and/or other albums.

* **Can I add image using FTP?**
Unfortunately, there is no such option. During the upload the thumbnails of the images are being created, whereas when using FTP the thumbnails do not get generated. The video thumbnails are required for most of the views. Thus using Media Upload is the only option.

== WordPress Add an gallery ==

### Step 1 Creating
**Starting to create our plugin**

* As a first step you should specify the name.

* Name. Provide a name for the image.

* Type - From this 5 gallery blocks you can choose only one.

* Displaying Content. Select the option which is more matches your website* 

* Per Page: Here you Can write quantity how many photos you want to see gallery in one page. Associated which galleries version did you select.

### Add unique Title to your photos, to complete your gallery

* **Title** - Important possibility to add title to each photo. The video title can be seen not in all views, but some of them have a place for it. Photo accomplished with title looks rich and perfect.

* **Description** - Wright some information about content

* **Link** - You can add a link to another page that is related. it will go to the link when you click the link button. You can add a link to another page that is related.

* **Open in New Tab** - Choose whether to publish "" or not.

* **Add images from Media Library** - You can select photos to use within the gallery which are currently uploaded to your WordPress Media Library by clicking on those and pressing " Insert into post " button. In addition, you can upload new files, navigating to the Upload Files tab on the left. Then press Select Files button, choose the photo and press Insert button. This way photo will be added to the gallery. 

**Add Photo URL** - You can also add Photos providing the absolute link to it. Press Add Image URL, provide the link and press Add button. This will be useful for adding photos to gallery without the need of downloading those and uploading again. **Add Image** - In addition to images, you can instead use photos within gallery from web site URL. You should click on the Add Image button, provide the link to the photo and press Add button. The photo associated with the web site image will be automatically displayed as a gallery. **Remove** - You can use Remove button.
Quantity photos in the gallery is unlimited. Quantity depends on your wishes. If you want to add one more photo, you must click on "Add New". By clicking the button below you will see sequence of windows. If you want to delete one of the creatures image clicking on the icon at the top of the bin will be deleted.

**If you are finished gallery creation and want to keep it, in the upper right side you'll see the SAVE button. Clicking on it will be Saved. After This in admin page you will see gallery photo created by you.**

### Manager.
**After you created gallery on the manager are 6 blocks.**

* **No** - Here is located the number Of your gallery.

* **Title** - Here you will see title wrote by you.

* **Quantity** - How many images you created.

* **Type** - Determine preferable type.

* **Shortcode** - Here you will see shortcode which you can copy and Paste [Juna_Photo_Gallery id="N"] where you want to see your plugin.

* **Templete Include** - Copy & paste <?php echo do_shortcode("[Juna_Photo_Gallery id="1"]");?> this code into a template file to include the slideshow within your theme.

* **Actions** - Edit. Press Edit you can make some changes, if you want to add or remove anything. * Actions - Delete: Press Delete created by you will delete.

* Below you will be able to change options.

### Lightbox Column

**Settings for Image Round**

* **Background Color** - Here you can select your favourite background color for column theme.
* **Column Border Width** - Define column gallery border width.
* **Column Border Style** - Select the border style for the column container.
* **Border Color** - Configure your preferable color for column border.
* **Border Radius** - Define the radius for the column border.
* **Box Shadow** - As you may know box-shadow is predominately a CSS3 property that allows you to display a shadow behind your column.
* **Shadow Color** - Set the shadow color for the column box.
* **Pading** - Space between containers image. Set the padding level between each item container.
* **Hover Rotate Show** - You can rotate the image. The change will also affect the image displayed within the gallery. Choose whether to show or not.
* **Hover Transparency Show** - This effect will create a hover transparency around your column container when you hover over it. Choose whether to show or not in your gallery.
* **Hover Zoom Show** - This effect will zoom your images when you hover over it. Choose whether to show or not in your gallery.

**Settings for Image**

* **Border Width** - Define the border width for images in the column box.
* **Border Style** - Choose the border style for images in the column box.
* **Border Color** - Define the border color for photos in the column box.
* **Border Radius** - Define image border radius for the column gallery.
* **Hover Transparency** - Set the level of transparency for the background of the image.

**Settings for Title**

* **Color** - Select color tune it with your image.
* **Text Shadow** - Set the shadow size for the text.
* **Font Size** - Determine title size of the letters of column gallery.
* **Font Family** - Specify the font family for the text in column.
* **Text Align** - Different among 4 position for the image title. Solid, Dotted, Dashed, Double.
* **Line style** - Choose the style to be applied to the line.
* **Line Width** - Set the width of line.
* **Line Color** - Choose a color for separation line.

**Settings for Description**

* **Description Color** - Set the color for the image description.
* **Font Size** - Specify the font size for the image description.
* **Font Family** - Choose the font family for the gallery description.
* **Text Align** - Choose the text position for gallery description. Please make sure it does not coincide with the title position avoiding overloading.

**Settings for Link**

* **Link Text** - You can change the “View More” button text and write something new.
* **Color** - Pick the link text color for the lightbox column view.
* **Font Size** - Set the font size for the View More button.
* **Font Family** - Specify the font family for the text for the link.

**Settings for Lightbox**

* **Lightbox Effect Type** - Select the effect which will be applied to the slideshow of images in the lightbox. There are 20 types of effects.
* **Lightbox Background Color** - Change the background color of gallery lightbox.
* **Transparency** - Specify the background transparency level for the lightbox.
* **Lightbox Content Width** - Lightbox max width Set unfix content max width.

**Settings for Popup Title**

* **Font Size** - Define the font size for the lightbox image title.
* **Font Family** - Define the font family for the lightbox image title.
* **Color** - Define the title color for the lightbox gallery.
* **Text Align** - Choose the text position for gallery title. Please make sure it does not coincide with the description position avoiding overloading.
* **Line Color** - Choose color for separation line.
* **Style** - Pick up one style for separational line, which will divide title from description.
* **Width** - Choose width size for separation line.

**Settings for Lightbox Description**

* **Description Show** - Select whether to see the description text or not in the lightbox.
* **Background Color** - Determine the color for description field. 
* **Text color** - Define font color for description text.
* **Font size** - Define the font size for the description.
* **Font family** - Specify font family for description used with image.
* **Text Align** - Choose the text position for gallery description.

**Settings for Lightbox Slider**

* **Autoplay Slideshow** - Select whether to see the slideshow or not in the lightbox gallery.
* **Slideshow Pause Time** - Set the time interval for the change of the galleries in milliseconds in the lightbox.
* **Icon Size** - Alter the size of the icon regardless of the lightbox (px).
* **Delete Icon Size** - Alter the size of the delete icon regardless of the lightbox.
* **Delete Icon Color** - Alter the color of the delete icon regardless of the lightbox.

**Choose Icons for Lightbox**

* **Icon** - assign a default icon in the slideshow. You can select icons from our 3 different beautifully designed sets.

* **Choose Icons for Slider** - assign a default icon in the slideshow. You can select icons from our 18 different beautifully designed sets for the gallery.
